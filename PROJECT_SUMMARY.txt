🚀 КАК ЗАПУСТИТЬ
═══════════════════════════════════════════════════════════════════════════════

1️⃣ Откройте PowerShell в папке проекта:
   cd "C:\Users\слава\Documents\GitHub\пивоо"

2️⃣ Установите зависимости:
   pip install -r requirements.txt

3️⃣ Создайте тестовые данные (опционально):
   python init_db.py

4️⃣ Запустите приложение:
   python run.py

5️⃣ Откройте в браузере:
   http://localhost:5000

🛠 ТЕХНИЧЕСКИЙ СТЕК
═══════════════════════════════════════════════════════════════════════════════

Язык: Python 3.8+
Веб-фреймворк: Flask 2.3.3
ORM: SQLAlchemy 3.0.5
Аутентификация: Flask-Login 0.6.2
Валидация: WTForms 3.0.1
Безопасность: Werkzeug 2.3.7
База данных: SQLite (легко заменить на PostgreSQL/MySQL)

Frontend:
  • HTML5
  • CSS3 (с переменными для тем)
  • Vanilla JavaScript (без зависимостей)
